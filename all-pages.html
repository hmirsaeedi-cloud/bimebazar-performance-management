One place to open every temporary page created for the performance management scaffold.

Roadmap Interactive 10-sprint roadmap from the PRD prioritization table. Login Temporary Supabase login page for the HR Admin account. Employee Profiles Manual profile creation with Jalali join date support. Employee Export Generate a filtered CSV report from employee profiles. Calendar Settings Update Jalali/Gregorian display, locale, and timezone preferences. Language + RTL Update Persian/English language and right-to-left display preferences. Form Builder Create and publish a form template with all question types. Form Versioning Create draft edit versions from existing form templates. RBAC Manager Sync Recompute automatic Manager roles from active direct reports. MPA Workflow Create MPA cycles and move agreements through approval steps. MPA History Capture, review, restore, and audit MPA version snapshots. Bulk Employee Import Paste Excel/CSV rows, preview validation, and create employees. Process Engine Create, configure, start, pause, and complete performance processes. Self-assessment Start employee self-assessment tasks and move them through manager review. Downward Routing Route manager evaluations through next-level manager and HRBP review. End-cycle Evaluation Create weighted evaluations and route them through NL, Head, and HRBP review. Mid-cycle Evaluation Create weighted checkpoint evaluations with manager and HRBP review. Side-by-side Evaluation Compare self-assessment and manager answers with gated score visibility. Performance Band Flags Auto-detect PIP and Promotion recommendations from submitted weighted scores. PD Chat Log development conversations and auto-attach them to evaluation records. Audit Log Inspect immutable compliance events, hash integrity, and exports. Notifications Create and manage in-app notification inbox items. Notification Preferences Configure channel settings, digest cadence, quiet hours, and audited overrides. Role Dashboards Inspect Employee, Manager, HRBP, and HR Admin dashboard views. HRBP Reports Generate aggregated HRBP analytics snapshots with workflow controls. Feedback Requests Create feedback requests with anonymous response thresholds and release guards. Process Form Instances Inspect locked form instances and move forms with required admin audit reasons. Individual Surveys Create targeted survey processes with locked form versions and audited responses. Promotion Workflow Create and approve career promotion cases. PIP Workflow Create hidden performance improvement plans and activate employee visibility after HRBP approval. Email Notifications Create, approve, queue, and track email delivery events.
