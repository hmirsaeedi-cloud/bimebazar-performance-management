BimeBazar Local Pages

One place to open every temporary page created for the performance management scaffold.

Roadmap Interactive 10-sprint roadmap from the PRD prioritization table. Login Temporary Supabase login page for the HR Admin account. Employee Profiles Manual profile creation with Jalali join date support. Employee Export Generate a filtered CSV report from employee profiles. Calendar Settings Update Jalali/Gregorian display, locale, and timezone preferences. Language + RTL Update Persian/English language and right-to-left display preferences. Form Builder Create and publish a form template with all question types. RBAC Manager Sync Recompute automatic Manager roles from active direct reports. MPA Workflow Create MPA cycles and move agreements through approval steps. Bulk Employee Import Paste Excel/CSV rows, preview validation, and create employees. Process Engine Create, configure, start, pause, and complete performance processes. Self-assessment Start employee self-assessment tasks and move them through manager review. Downward Routing Route manager evaluations through next-level manager and HRBP review. End-cycle Evaluation Create weighted evaluations and route them through NL, Head, and HRBP review. Mid-cycle Evaluation Create weighted checkpoint evaluations with manager and HRBP review. PD Chat Log employee and manager development conversations with workflow controls.
